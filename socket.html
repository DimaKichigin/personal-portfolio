Онлайн чат

Онлайн чат в реальном времени.
В чате можно создавать комнаты для общения, подключать других участников, выходить из комнаты,
а также отправлять смайлики вместе с текстом.
Для создания сервера были использованы Node.js и Express.js.
В качестве передачи данных и обработки событий использовался Socket.io для установления соединения с клиентом.
Для фронтенда были использованы React и снова Socket.io.
Развертывание онлайн-чата было выполнено на Netlify и Render.
